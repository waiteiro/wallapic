## 🎯 Cambios Principales

### Sistema de Almacenamiento Híbrido
- Nuevo archivo `storage-manager.js` que detecta automáticamente si hay sesión
- Sin sesión: todos los datos en localStorage (modo offline)
- Con sesión: todos los datos en Supabase (modo cloud sincronizado)

### Esquema de Base de Datos Actualizado
- Tabla `users` con columnas `avatar` y `bio`
- Tabla `entries` con columna `is_public` (privadas por defecto)
- Tabla `used_words` para palabras del diccionario
- Tabla `pinned_images` para imágenes marcadas
- Políticas de seguridad RLS correctas incluyendo UPDATE

### Funcionalidad de Entradas
- Entradas privadas por defecto al guardar
- Botón "Hacer Pública" para usuarios autenticados
- Botón toggle para cambiar entre público/privado
- Feed público muestra SOLO entradas marcadas como públicas
- Badge "Público" en archivo personal

### Sistema de Autenticación Mejorado
- `window.currentUser` exportado globalmente
- Carga de perfil completo al iniciar sesión
- Recarga automática de datos después de login/registro
- Logout limpia sesión y recarga página
- Avatar con compresión automática (10MB → KB)

### Sincronización de Datos
- Entradas: privadas y públicas en Supabase
- Palabras aprendidas: sincronizadas
- Imágenes pineadas: sincronizadas (máx 5)
- Perfil completo: avatar, bio, stats

## 🔧 Breaking Changes

- Requiere ejecutar nuevo SQL schema en Supabase
- Los datos antiguos en localStorage NO se migran automáticamente
- Usuarios deben crear cuenta para usar funciones cloud
